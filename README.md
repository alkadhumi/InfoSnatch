# InfoSnatch

InfoSnatch is a powerful tool designed to gather real-time information about users, including their IP address, live location, device details, and even capture an image from their camera (with permission).

## Features
- ‚úÖ **IP Address Tracking** - Retrieves the public IP of the user.
- ‚úÖ **Live Geolocation** - Fetches approximate location using IP-based geolocation.
- ‚úÖ **Device Information** - Captures browser and OS details.
- ‚úÖ **Camera Access** - Captures an image from the user's webcam.

## Installation
```bash
git clone https://github.com/yourusername/InfoSnatch.git
cd InfoSnatch
pip install -r requirements.txt
```

## Create a .env file and add your Telegram Bot Token and Chat ID
```bash
TOKEN=your_telegram_bot_token
CHAT_ID=your_telegram_chat_id
```

## Run the application
```bash
python app.py
```
